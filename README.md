# 内核算法库

## 简介

这是一个开源的 **内核算法库**，使用 **C++** 实现，旨在为开发者提供高效的算法实现，解决常见的计算和优化问题。项目支持包括CAM加工路径计算等，所有实现都经过精心优化，适合大规模数据计算。

本项目基于 **Google Test** 测试框架进行单元测试，确保每个模块都经过全面的验证。

通过优化的内核算法，可以广泛应用于制造业、数据科学、机器学习等领域，特别是在 **CAM（计算机辅助制造）** 系统中的刀具路径计算、3D建模与分析等方面，提供强有力的算法支持。

## 特性

- **高效的C++实现**：针对内核算法进行性能优化，适用于大规模数据集。
- **经典算法集合**：包括排序、图算法、动态规划等常用算法，涵盖广泛领域的基础算法。
- **支持并行计算**：提供并行化优化，适合多核处理，极大提升计算效率。
- **Google Test集成**：所有功能都有单元测试保障，确保代码稳定性。
- **跨平台支持**：兼容主流操作系统（Linux, macOS, Windows），无缝运行于不同环境。
- **适用场景广泛**：支持 **CAM** 系统中的刀具路径计算、曲线简化与优化、3D模型分析、图像处理等应用。

## 安装

### 依赖项

在安装项目之前，请确保系统中已安装以下工具和库：

- **C++ 编译器**（例如 GCC 或 Clang）
- **CMake**（用于构建项目）
- **Google Test**（用于单元测试）

### 安装 Google Test

如果你还没有安装 Google Test，你可以通过以下方式安装：

#### 使用 Git 克隆 Google Test

```bash
git clone https://github.com/google/googletest.git
cd googletest
cmake CMakeLists.txt
make
sudo make install
```
### 克隆本项目

```bash
git clone https://github.com/your-username/kernel-algorithms.git
cd kernel-algorithms
```

### 使用 CMake 构建项目

首先，创建构建目录：

```bash
mkdir build
cd build
```

然后，使用 CMake 配置和生成构建文件：

```bash
cmake ..
```

构建项目：

```bash
make
```

### 运行单元测试

运行所有测试：

```bash
make test
```

或者，你可以使用 `ctest` 工具来运行测试：

```bash
ctest
```

## CAM 精确刀具路径计算

本库针对 **CAM（计算机辅助制造）** 系统的刀具路径计算进行了多项优化，提升了计算效率与路径精度。以下是一些关键模块：

✔ **提升制造精度**：优化刀具路径生成算法，精确控制刀具在加工过程中的运动轨迹，确保产品的尺寸和形状精度。

✔ **低成本集成**：简化计算过程，降低系统计算成本，同时保证高效的路径规划和数据处理。

✔ **加速研发**：通过引入高效算法和并行计算优化，缩短了研发周期，支持更快的产品迭代。

✔ **优化工作流**：通过路径优化算法减少加工时间，提升整体生产效率，助力制造过程中的生产自动化。

### 2D曲线处理

* **轮廓曲线简化**：通过对轮廓曲线的简化，使得刀具路径生成更加高效，减少不必要的计算和时间开销。
* **多边形偏移**：在2D空间内，优化多边形的偏移操作，确保刀具路径能够精确匹配到目标形状，且具备良好的加工效率。
* **开槽路径优化**：优化开槽过程中的刀具路径，避免重复路径，提高生产效率。

### 3D模型分析

* **三角网格切割**：通过优化三角网格切割算法，将大规模的3D模型分割成适合加工的部件，减少计算量并提升处理效率。
* **投影轮廓计算**：对复杂3D模型进行投影处理，生成精确的刀具路径，适用于复杂曲面和不规则形状的加工。
* **测地曲线生成**：生成3D模型表面上的测地曲线，用于刀具路径的精确规划，确保刀具在曲面上的运动更加平滑。
